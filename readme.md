# eating-snake

#### Introduction

A little snake game written in C.


#### How to install

1.  Running 'make' command.
2.  Move executable file to executable dir.
3.  That's all.


#### How to use

1.  Enter 'eating-snake' to launcher game.
2.  Press 'H' for help when game is running.
