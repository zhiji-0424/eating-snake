# EatingSnake

#### 介绍
使用C语言编译的贪吃蛇小游戏

#### 软件架构
目前仅支持 ARM64-Linux

#### 安装教程

1.  运行命令[make]
2.  将生成的可执行文件放目录[~/.local/bin/]
3.  完成

#### 使用说明

1.  运行命令[eating-snake]
2.  按键[H]查看帮助

#### 参与贡献

1.  Fork 本仓库
2.  新建 Feat_xxx 分支
3.  提交代码
4.  新建 Pull Request
